# 🌿 Pomodoro Study Timer

> *Focus. Grow. Rest. Repeat.*

A beautiful nature-themed Pomodoro timer that makes studying feel like tending a forest.
Built with **zero dependencies** — pure HTML, CSS, and JavaScript.

<br>

## 🌐 [→ Open Live App](https://anant-083.github.io/Pomodoro_Study_Timer/)

<br>

---

## 🌳 The Idea

Every time you complete a 25-minute focus session, a tree fully grows and joins your personal forest.
Dismiss the timer early? Your tree withers. Stay focused — build your forest, one session at a time.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🕐 **Pomodoro Cycle** | 25 min focus → 5 min break, auto-starts |
| 🌱 **Live Tree Growth** | Tree grows through 5 stages as you study |
| 🥀 **Wither on Dismiss** | Leave early and watch your tree wither |
| 🌲 **Your Forest** | Completed sessions build a persistent forest |
| 🎵 **Ambiance Sounds** | Rain, Ocean, Wind, Study Tones, or Silence |
| 🔊 **Volume Control** | Adjust ambiance to your comfort |
| 📊 **Study Stats** | Track today's sessions, total count & daily streak |
| 🌙 **Day / Night Theme** | Auto-switches based on your real clock time |
| 🌧️ **Rain Animation** | Live animated raindrops on the background |
| 💬 **Quotes** | A fresh motivational quote every session |

## 🎵 How Sounds Work

All sounds are **generated in real-time** using the Web Audio API — no audio files are downloaded.

- 🌧️ **Rain** — Water droplet synthesis + soft background mist
- 🌊 **Ocean** — Slow wave cycles with layered noise buffers
- 🌬️ **Wind** — LFO-modulated filtered noise
- 🎵 **Study Tones** — Pentatonic scale chimes + binaural beats
- 🔇 **None** — Pure silence
